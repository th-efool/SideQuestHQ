# Discord / Telegram

## Target communities

- Study servers, exam prep servers, “self improvement” servers
- Telegram: student channels + coaching channels

## Execution

- Build a list of 50 communities (link + rules + mod contact)
- Start with “contribute for a week” before pitching in that community

## Value-first posts (examples)

- Free “14-day playlist plan” for a popular playlist
- Weekly “study sprint” template + checklist
- Share curated “best X playlist” + how to finish it

## Outreach scripts

- Mods: “Can I run a free 7-day playlist challenge in your server? I’ll do the planning + daily prompts.”
- Users: “If you drop a playlist link, I’ll turn it into a daily plan.”

## Challenge format (7 days)

- Day 0: pick playlist + goal
- Days 1–6: daily lesson + check-in
- Day 7: share results + collect testimonials (real users only)
