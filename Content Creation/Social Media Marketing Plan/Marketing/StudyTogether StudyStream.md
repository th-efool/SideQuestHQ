# StudyTogether / StudyStream

## Target

- People already doing public “study sessions” / accountability livestreams
- Students prepping for exams; self-learners following long playlists

## Where to post

- StudyTogether / StudyStream communities
- Adjacent Discord study servers

## Message (templates)

- “I built a tool that turns any YouTube playlist into a daily course plan + streak. Looking for feedback.”
- “If you’re overwhelmed by long playlists, paste the link → it schedules it into 20–30 min/day.”

## Weekly plan

- Mon: 20 DMs to community mods / server owners
- Tue–Thu: 5 value-first posts + 20 helpful replies
- Fri: share 3 user wins + ask for feedback
- Sat/Sun: run a 7-day “finish 1 playlist” challenge

## Assets to prepare

- 1-min demo video
- 3 screenshots (paste link → course → day view)
- Landing page with UTM: `?utm_source=studytogether`
