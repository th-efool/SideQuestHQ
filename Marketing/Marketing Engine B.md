# 1. Hijack Existing Learning Intent (highest ROI)

Your users already exist. They’re here:

- YouTube comment sections
- Reddit threads
- Discord communities

---

## 🎯 Exact play (don’t overcomplicate)

Find videos like:

```
"Full Python course 12 hours"
"Complete Machine Learning course"
```

Go to comments. You’ll see:

```
"Saving this for later"
"Will watch this someday"
"Too long but looks good"
```

These are **pain signals**.

Reply:

```
"I kept saving stuff like this and never finishing.
Built something that splits it into daily 15-min lessons so you actually complete it.
If you want I can send it."
```

Not spammy. Not pitchy.

You’re matching their exact thought.

---

# 2. Short-form Content Engine (your main growth driver)

Platforms:

- TikTok
- Instagram Reels
- YouTube Shorts

---

## 🎬 What you actually post

Not features. Not UI.

Post **transformation loops**:

```
"This is a 18-hour course.
I turned it into 12 days, 15 mins/day.

Day 1 → done
Day 3 → still going
Day 7 → didn’t expect this
Day 12 → finished"
```

Then show:

```
progress streak
completion bar
micro-lessons scrolling
```

---

## Why this works

You’re triggering:

```
aspiration ("I want this")
+ relatability ("this is me")
+ proof ("it worked")
```

# 3. “Finish What You Start” Challenge

Make this a public loop.

---


# 4. Micro Communities > Big Audience

Ignore scale initially.

Go here:

- Reddit (r/learnprogramming, r/productivity)
- niche Discord servers
- Telegram groups

---

## Your message

Not:

```
"Check out my app"
```

Instead:

```
"I couldn’t finish long courses so I built something that forces me to do 20 mins/day.
Anyone want to try?"
```

This pulls, not pushes.
