## North star

- Goal: consistent weekly user signups + retention loops
- Positioning (multiple angles)
    - “Join or start a learning squad” (community + accountability)
    - “Gamified learning streaks for real skills” (levels, quests, leaderboards)
    - “Stop saving playlists — actually finish them” (execution layer for long-form learning)
    - “Cohorts without cringe” (guided sprints + challenges, not ‘make a course’)
- CTA (choose per channel)
    - Join a challenge / squad
    - Start a 7-day sprint
    - Drop a link → get a daily plan + squad challenge
